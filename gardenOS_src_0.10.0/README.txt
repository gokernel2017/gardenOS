
 GARDEN OPERATING SYSTEM:

 A x86 (32 bits) Operating System in C.

   compile_windows_gcc.bat

 TO COMPILE IN LINUX TYPE:
   ./compile_linux_gcc.sh

   qemu -kernel gardenOS_kernel
